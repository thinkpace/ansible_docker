thinkpace.ansible_docker
=========

This is a very simple role to install docker.

Dependencies
------------

This role is tested on Ubuntu 20.04 (Focal Fossa).

Example Playbook
----------------

This role can be used like following:

```
    - hosts: servers
      roles:
         - { role: thinkpace.ansible_docker }
```
